Character Palace!


You will be given a random number at the start of the game

There are four Characters below. By clicking on the Character you will add a specific amount of points to your total score.

You win the game by matching your total score to random number, you lose the game if your total score goes above the random number.

The value of each Character is hidden from you until you click on it

Each time the game starts, the game will change the values of each Character
